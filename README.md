# Line Clipping Algorithm - Computer Graphics

Project implementasi algoritma kliping garis untuk memenuhi tugas mata kuliah Grafika Komputer.

## Fitur
- [cite_start]**Cohen-Sutherland**: Menggunakan region code 4-bit[cite: 206].
- [cite_start]**Liang-Barsky**: Menggunakan persamaan parametrik yang lebih efisien[cite: 393].
- **Visualisasi Interaktif**: Menggunakan Matplotlib dengan skema warna modern.
